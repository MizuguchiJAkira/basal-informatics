# ── Version posture ──────────────────────────────────────────────────────
# Web + security-relevant packages are pinned with the compatible-release
# operator (``~=``) so patch / minor versions keep flowing for CVE fixes,
# but a future major-version transitive bump can't drag us into breakage
# or regression silently. Non-security packages stay on ``>=`` so the
# usual dependency resolution still has room.
#
# To refresh: pip-audit (or pip list --outdated) → bump the locked
# numbers below → re-run the full test suite → commit.

flask-wtf~=1.2
# Werkzeug + Jinja2 are flask dependencies; pinning them explicitly so a

# YAML loader for Stage 7 reference datasets — pinned because we use it
