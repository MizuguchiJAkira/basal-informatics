# ⚠️ Sample Nature Exposure Report

`nature_exposure_sample.pdf` in this directory is a **sample report
generated against the synthetic demo dataset** (`demo/` in this repo —
the hypothetical "Edwards Plateau Ranch" scenario). It is published to
show the structure, formatting, and citation conventions of a real
Basal Nature Exposure Report — not to present fabricated data as real.

The cover page of the PDF carries a matching disclaimer.

Any reference to this PDF in outreach, pitches, or documentation
should carry the "sample, synthetic demo data" framing forward.

Production Nature Exposure Reports are generated per-parcel from
actual customer submissions and are not published in this
repository.
